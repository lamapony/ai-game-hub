# Grill + Bar Party Upgrade — ТЗ для исполнителей

Дата: 2026-07-03
Целевой артефакт: `park-ai-games.html` (standalone ~880 строк, Tailwind + FontAwesome,
клиентское, камера/микрофон). Файл живёт вне этого репозитория; инструкции привязаны
к структурным ориентирам (`#mode-1…4`, `switchMode`, grid карточек), не к номерам строк.

Контекст вечера: две фазы — 🔥 гриль (17:00, Grønningen Nordvest, outdoor) и
🍸 бар (20:00, Viggos Bar, Nørrebro, indoor). Гости — взрослые умные люди, тон —
остроумный, слегка саркастичный, двойные смыслы допустимы.

ЖЁСТКОЕ ПРАВИЛО: ничего не удалять. Старые 4 игры (AI Детектив, AI Скавенджер,
AI Чародей, Битва Подписей) и их промпты остаются полностью рабочими («Классика Парка»).
Только дополняем.

## 1. Архитектура изменений

### 1.1 Система фаз

```js
const PARTY_PHASES = {
  classic: {
    label: "Классика Парка", emoji: "🌳",
    accent: "emerald",
    headerGradient: "from-emerald-600 to-teal-700",
    aiContext: "" // старое поведение без изменений
  },
  grill: {
    label: "Гриль", emoji: "🔥",
    accent: "orange",
    headerGradient: "from-orange-600 via-red-600 to-rose-700",
    aiContext: `ЛОКАЦИЯ: гриль-сбор в Grønningen Nordvest, Копенгаген, 17:00. Ветер, возможен дождь, все делают вид, что так и задумано. РЕКВИЗИТ СЦЕНЫ: огонь, дым, щипцы, решётка, мясо, овощи, фольга, одноразовые тарелки, борьба со стихией. Шути про степень прожарки, датскую погоду, коллективное приготовление и людей, которые "просто посмотрят, как ты жаришь".`
  },
  bar: {
    label: "Бар", emoji: "🍸",
    accent: "amber",
    headerGradient: "from-violet-950 via-purple-900 to-amber-900",
    aiContext: `ЛОКАЦИЯ: Viggos Bar, Nørrebro, после 20:00. Тёплый свет, бокалы, коктейли, тосты, вечерняя честность. Публика согрелась и осмелела. Шути про сомелье-снобизм, барную философию, красивые названия коктейлей и то, как быстро "по одной" превращается в "ну ладно, ещё по одной".`
  }
};

let currentPhase = localStorage.getItem("partyPhase") || "classic";

function setPhase(phase) {
  currentPhase = phase;
  localStorage.setItem("partyPhase", phase);
  document.body.className = document.body.className
    .replace(/phase-\w+/g, "").trim() + " phase-" + phase;
  renderPhaseUI(); // хедер-градиент, бейджи карточек, dim нерелевантных
}

function withPhaseContext(prompt) {
  const ctx = PARTY_PHASES[currentPhase].aiContext;
  return ctx ? ctx + "\n\n" + prompt : prompt;
}
```

Правило неразрушения: старые 4 режима не знают о фазах. Единственное касание — их
промпты проходят через `withPhaseContext()`. В `classic` контекст пустой → байт-в-байт
старое поведение.

### 1.2 UI

1. Хедер: сегмент-переключатель `🌳 | 🔥 | 🍸` справа от заголовка. Градиент хедера —
   из `PARTY_PHASES[phase].headerGradient`.
2. Карточки игр: атрибут `data-phases="grill"` / `"bar"` / `"grill,bar,classic"`.
   Нерелевантные фазе карточки — `opacity-40 grayscale` + бейдж «не в тему фазы»,
   но остаются кликабельными.
3. Цвета: гриль — `orange-600`/`red-700`, акцент `amber-500` (угли); бар —
   `violet-950`/`amber-400` (неон). Классика — emerald, как было.
4. Иконки: `fa-fire`, `fa-martini-glass-citrus`.

### 1.3 Очки

Единый Party Score, каждая запись лидерборда — с тегом фазы:

```js
// Было: addScore(name, points). Стало (обратно совместимо):
```

## 2. Новые геймплей-концепции (8)

### 🔮 2.1 «Гриль-Оракул» — Grill → Transition (vision + prophecy)

Одной строкой: AI читает судьбу по подпалинам на еде и выдаёт 3 проверяемых
пророчества на барную фазу.

Loop:
1. Игрок фотографирует готовый (или сгоревший) кусок с решётки.
2. Vision-AI в роли пиромантки-шарлатанки: предмет, вердикт прожарке, пророчество
   по узору подпалин + ровно 3 проверяемых предсказания на вечер.
3. Пророчества сохраняются в localStorage с именем игрока.
4. В баре — кнопка «Проверить пророчества»: зал подтверждает/опровергает каждое,
   AI пишет вердикт судьбы. Сбывшееся = очки владельцу, несбывшееся = скептикам.

Почему запомнится: единственная игра, физически связывающая две локации; зал весь
вечер следит за исполнением пророчеств. Слот: 15 мин гриль + 10 мин бар.

### 🍢 2.2 «Щипцы Правды» — Grill (audio)

Одной строкой: кто держит щипцы — тот на детекторе лжи; переворачивать мясо нельзя,
пока не ответил на вопрос AI.

Loop: щипцы = артефакт → AI задаёт вопрос (3 уровня остроты) → ответ 10–20 сек на
микрофон → Whisper → AI-судья: честность / уклончивость / артистизм → очки, публичный
комментарий уклонений → щипцы передаются со следующим куском.

Почему запомнится: механика встроена в реальное дело — щипцы всё равно ходят по рукам.
Слот: фоновая, весь гриль, 2 мин на ход.

### 🕵️ 2.3 «Дымовая Завеса» — Grill → Bar (social deduction)

Одной строкой: каждый получает тайную абсурдную миссию на гриль, а в баре зал голосует,
кто что выполнял.

Loop: на гриле каждый тайком получает миссию от AI (3 тира сложности: 5/10/15 очков) →
выполняет, маскируя под обычное поведение → в баре список миссий без имён → голосование
«кто какую выполнял» → очки: выполнил и не спалился = джекпот; вычислил чужую = очки
детективу → AI пишет recap-роаст.

Почему запомнится: Among Us без экрана; reveal в баре — гарантированный ор.
Слот: 2 мин раздача, фон на гриле, 15 мин reveal.

### 🥒 2.4 «Натюрморт: Выживание» — Grill (vision, команды)

Одной строкой: AI выдаёт абсурдный заголовок, команды строят из еды и утвари
драматическую сцену, AI-искусствовед оценивает как лот аукциона.

Loop: AI генерирует заголовок («Последний огурец покидает тонущую лодку») → команды
5 мин строят инсталляцию из еды/фольги/щипцов → фото → vision-AI как арт-критик
Sotheby's: композиция, драматургия, работа с материалом (бонус за дым/обугленность),
цена лота в DKK, название для каталога → победа по очкам, зрительский голос — тайбрейк.

Почему запомнится: обратная «Битва Подписей» — вы строите реальность под заголовок AI.
Слот: 20 мин (2 заголовка по 2 команды).

### 🥂 2.5 «Синдикат Тостов» — Bar (audio)

Одной строкой: AI выдаёт жанр тоста и 3 контрабандных слова — нужно провезти их
через речь, а публика ловит.

Loop: AI выдаёт жанр (нуар, питч инвесторам, оправдательная речь в суде, TED talk про
очевидное) + 3 слова («карбюратор», «фьорд», «неоднозначно») → тост 30–60 сек, зал
слушает и ловит слова → Whisper → AI-судья: жанр + гладкость провоза каждого слова →
очки говорящему за жанр и непойманные слова, слушателям — за пойманные.

Почему запомнится: тосты в баре и так будут — игра превращает ритуал в дуэль.
Слот: 20–25 мин, 6–8 тостов.

### 🍷 2.6 «Сомелье-Шарлатан» — Bar (vision + social guessing)

Одной строкой: AI-сомелье с фиктивным дипломом описывает анонимные фото напитков и
составляет психопортрет владельца — бар угадывает, чей бокал.

Loop: все тайно фотографируют напиток → vision-AI: дегустационные ноты (выдуманные,
но убедительные), уровень претенциозности 1–10, психопортрет владельца → портреты
зачитываются, зал голосует «чей напиток» → очки угадавшим; владельцу — если не угадали;
бонус за самый громкий ор.

Почему запомнится: двухслойный роаст — AI жарит напиток, компания через него — владельца.
Слот: 20 мин на 8–10 напитков.

### 📦 2.7 «Контрабанда» — Bar (фоновая social)

Одной строкой: каждому — тайная нелепая фраза, которую надо органично провезти через
живой разговор; поймали — конфисковали.

Loop: AI выдаёт фразы («я в целом доверяю уткам») → 20 мин обычных разговоров →
услышал подозрительное — кричишь «Контрабанда!» → спорный случай: обвинённый повторяет
контекст на диктофон, AI судит органичность 1–10 → провёз до таймера = джекпот; поймал =
очки; ложное обвинение = штраф.

Почему запомнится: игра живёт внутри настоящих разговоров, делает подозрительной саму
речь. Слот: фоновая, 20–30 мин.

### 🚨 2.8 «Перекрёстный Допрос» — Bar, финал (audio + callback)

Одной строкой: AI-следователь допрашивает пары порознь о событиях гриль-фазы и
вскрывает противоречия в показаниях.

Loop: пара «подельников» с гриля → AI генерирует 4 вопроса о мелкой конкретике вечера
(«что сгорело первым?») → каждый отвечает отдельно на диктофон → Whisper → AI сверяет:
расхождения, прочность алиби, нуар-вердикт → очки паре за согласованность; залу — за
предсказание, где разойдутся.

Почему запомнится: превращает весь вечер в общее дело с уликами; проверяет память о
реально прожитом вечере. Слот: 20 мин, 3–4 пары, финал перед титулами.
